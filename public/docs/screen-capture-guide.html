How to Capture Screenshots

  1. Navigate to each URL listed below in the v0 preview panel
  2. Use Win + Shift + S (Windows) or Cmd + Shift + 4 (Mac) to capture
  3. For full-page screenshots: Right-click > Inspect > Ctrl+Shift+P > Type "screenshot" > Select "Capture full size screenshot"
  4. Check off each screen as you capture it
