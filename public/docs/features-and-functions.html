1. System Overview

The SGC Portal is a comprehensive digital management system designed for the Solicitor General's Chambers of Barbados. It provides a centralized platform for managing correspondence and contracts between government ministries, departments, and agencies (MDAs).

Key Objectives

  • Digitize and streamline correspondence management workflows
  • Centralize contract registration and tracking
  • Provide real-time visibility into document status and processing
  • Enable comprehensive reporting and analytics
  • Maintain complete audit trails for compliance

System Architecture

Technology Stack

  • Frontend: Next.js 16 with React 19
  • Styling: Tailwind CSS v4 with shadcn/ui components
  • Authentication: Role-based access control
  • AI Assistant: Ask Rex intelligent chatbot

5. Registers

The Registers module provides comprehensive management of correspondence and contracts received by the Solicitor General's Chambers.

5.1 Correspondence Register

Purpose

Track and manage all incoming and outgoing correspondence from government ministries, departments, and agencies.

Features

  • Search & Filter: Search by reference number, subject, or MDA with status filters
  • Add New Correspondence: Register new correspondence with full details
  • Status Tracking: Monitor status (Pending, Under Review, Approved, Completed, Archived)
  • Priority Levels: Assign priority (Low, Normal, High, Urgent)
  • Document Attachments: Upload and manage supporting documents
  • Export: Export correspondence data for external use

Data Fields

Field Description
Reference NumberUnique identifier (e.g., COR-2026-0001)
Date ReceivedDate correspondence was received
SubjectBrief description of the correspondence
Originating MDAMinistry/Department/Agency that sent the correspondence
CategoryClassification of correspondence type
StatusCurrent processing status
PriorityUrgency level
Assigned ToStaff member responsible for processing

URL: /management/correspondence-register

5.2 Contracts Register

Purpose

Manage government contracts submitted for legal review and approval by the Solicitor General's Chambers.

Features

  • Search & Filter: Search by contract number, subject, or MDA with nature/status filters
  • Add New Contract: Register new contracts with comprehensive details
  • Status Tracking: Monitor contract status through approval workflow
  • Contract Types: Support for New, Renewal, Amendment, and Supplemental contracts
  • Value Tracking: Record and track contract values
  • Document Management: Attach contract documents and supporting materials

Data Fields

Field Description
Contract NumberUnique identifier (e.g., CON-2026-0001)
Date ReceivedDate contract was received
SubjectContract description
Originating MDAMinistry/Department/Agency submitting the contract
Nature of ContractGoods, Works, or Consultancy/Services
CategoryContract classification
Contract TypeNew, Renewal, Amendment, Supplemental
Contract ValueMonetary value of the contract
ContractorName of the contracting party
StatusCurrent processing status

URL: /management/contracts-register

13. Ask Rex AI Assistant

Ask Rex is an intelligent AI assistant integrated into the SGC Portal to help users navigate the system and find information quickly.

Appearance

Rex appears as a green floating button in the top-right corner of all management pages. Click to open the chat interface.

Capabilities

  • Report Generation: Request correspondence or contract reports for specific time periods
  • Search Assistance: Help finding specific records by reference number, subject, or MDA
  • Statistics Queries: Ask about system statistics and metrics
  • Navigation Help: Get guidance on using portal features
  • General Questions: Answer questions about SGC processes and procedures

Example Queries

  • "Generate a report on contracts submitted this week"
  • "Show me correspondence from Ministry of Finance"
  • "What are the current system statistics?"
  • "How do I add a new contract?"
  • "List all pending items"

Response Display

Rex displays responses with:

  • Formatted text responses
  • Data tables for reports
  • Statistics cards
  • Searchable result lists
